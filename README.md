# 🚀 Polymath Engineer Portfolio — Naufal Auzan Ramadhan

A high-performance, dark-mode, and futuristic 3D portfolio website built with **React.js** and **Three.js**. This project showcases the intersection of hardware engineering, robotics, and software development.

## 👨‍💻 About The Author

**Naufal Auzan Ramadhan** is a Computer Engineering student at **IPB University** who bridges the gap between mechanical fabrication, embedded systems, and modern web software. Dedicated to building "At Every Layer" — from welded chassis to real-time cloud-integrated dashboards.

### 🛠️ Core Expertise

- **Hardware**: Mechanical Fabrication (Welding, Lathe, Fabrication), PCB Design, & Hardware Integration.
- **Embedded Systems**: Arduino, ESP32, Raspberry Pi, & RTK-GPS integration.
- **Software**: React.js, JavaScript, Three.js (3D Visualization), & Python (Machine Learning/OpenCV).
- **Robotics**: Autonomous UGV navigation, Teleoperation systems, & Agricultural Automation.

---

## ✨ Features

- **3D Hero Section**: Interactive 3D component visualization using Three.js / React Three Fiber.
- **Futuristic Terminal**: A custom simulated terminal experience reflecting a developer's environment.
- **Mobile Responsive**: Fully adaptive layout for mobile, tablet, and desktop viewports, ensuring a premium experience on any device.
- **Resume Integration**: Dynamic education and work experience timelines with downloadable ATS-friendly CVs and Certificates.
- **Projects Showcase**: categorized grid displaying robotics, embedded, and software engineering projects.

---

## 📦 Tech Stack

- **Frontend**: React.js (Hooks, Functional Components)
- **3D Engine**: Three.js & React Three Fiber
- **Styling**: Vanilla CSS (Custom Variable Theme System)
- **Deployment**: Optimized for Vercel / Netlify
